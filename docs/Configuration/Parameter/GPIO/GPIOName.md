# Parameter: GPIO Name

|                   | WebUI               | Config.ini
|:---               |:---                 |:----
| Parameter Name    | GPIO Name           | IOx: 13. parameter
| Default Value     | empty -> `gpioX`    | empty


## Description

Name which is used as MQTT topic name


!!! Note
    If parameter is empty, default name (gpioX) is used, e.g. gpio1
