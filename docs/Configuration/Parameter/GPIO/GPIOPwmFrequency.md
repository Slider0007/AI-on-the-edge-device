# Parameter: GPIO PWM Frequency

|                   | WebUI               | Config.ini
|:---               |:---                 |:----
| Parameter Name    | GPIO PWM Frequency  | IOx: 4. parameter
| Default Value     | `5000`              | `5000`
| Input Options     | `5 .. 1000000`      | `5 .. 1000000`
| Unit              | Hertz               | Hertz



## Description

GPIO PWM frequency (only for PWM controlled GPIO modes)
