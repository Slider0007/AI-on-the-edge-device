# Parameter: LED Quantity

|                   | WebUI               | Firmware
|:---               |:---                 |:----
| Parameter Name    | LED Quantity        | quantity
| Default Value     | `1`                 | `1`
| Input Range       | `1` .. &infin;      | `1` .. &infin;


## Description

Quantity of smart LEDs which are connected as daisy chain (e.g. LED stripe)
